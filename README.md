# Fogón & Arte — Proyecto Front End

Este proyecto es una demostración de una página web desarrollada para la **segunda prueba del ramo de Front End**. Se trata de una tienda de artículos de cocina ("Fogón & Arte") que pone a prueba conocimientos avanzados de HTML5, CSS3 y JavaScript moderno.

## Características principales
- **Manipulación del DOM**: Generación dinámica de productos, filtrado en tiempo real y gestión del carrito de compras sin recargar la página.
- **JavaScript Moderno**: Uso de módulos IIFE, manejo centralizado de errores, validación de formularios y almacenamiento local (LocalStorage).
- **Diseño Responsivo**: Interfaz elegante y funcional adaptada a distintos dispositivos.
- **Tema Dinámico**: Soporte para modo claro y oscuro con persistencia de preferencias.
- **Interactividad**: Notificaciones visuales (Toasts), modales detallados y un panel lateral para el carrito.

## Estructura del Proyecto
- La versión definitiva y más refinada del proyecto se encuentra en la carpeta `PFE/`.
- `PFE/index.html`: Estructura semántica de la aplicación.
- `PFE/styles.css`: Estilos visuales con variables de CSS y animaciones.
- `PFE/index.js`: Lógica de negocio y manipulación dinámica de la interfaz.

## Prompts utilizados para revisión del proyecto
- Revisa los archivos adjuntos en base a la seguridad de la manipulación y almacenamiento de los datos obtenidos de los usuarios.
- Comprueba la eficiencia de la estructura de los datos que se almacenan en la página y marca lo que podría ser mejorado y como se pueda mejorar.
- Checkea que las imagenes y la adición de productos funcione y que no existan posibles loops los cuales puedan causar un error en la página.

## Reflexión de los prompts
- Al tener un recurso tan potente como es la IA nos sirve para validar que lo que hemos trabajado cumpla con los estandares de optimización y seguridad
  que hemos trabajado, logrando así obtener un producto final limpio y eficiente para presentar a posibles clientes.
